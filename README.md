# README

## Introduction
It's a graduation project, using for RSS/ATOM Subscripting and reading.

## Requirements
* ruby 2.6.0
* rails ~> 5.2.2

## Commands to run
* bundle install
* SIDEKIQ_THREADS=1 bundle exec sidekiq // SIDEKIQ_THREADS depends your server
* rails db:migrate  // rails db:setup 
* rails server
